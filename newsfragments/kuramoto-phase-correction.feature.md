Added `core.kuramoto.phase_correction` — forced-Kuramoto phase correction toward a baseline phase reference (`dθ/dt = K·sin(θ_ref − θ)`). Replaces a candidate "decentralized reset wave" pseudo-method that had eight class-of-bug problems (non-modular phase distance, mislabeled "energy" that was actually velocity, hardcoded trust-me booleans, magic-number tolerance, decorative parameter naming). Honest implementation: modular `wrap_to_pi`, circular phase distance, separate velocity / potential / max-error, derived convergence verdict, documented stability bound `K·dt < 2`. 18 tests including Hypothesis fuzz on convergence rate and a negative control falsifying the stability bound.
