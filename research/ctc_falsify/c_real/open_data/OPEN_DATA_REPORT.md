## Honest limitation (the designed outcome, not a defect)

Coarse archive metadata confirms spikes / LFP / size / subjects but
**cannot** confirm ≥2 areas, trial structure, or an INDEPENDENT routing
label (attention / opto / microstim). The frozen C-real preregistration
makes the independent label **mandatory**; it is never metadata-
inferable, so every metadata-only candidate stays
`UNKNOWN_NEEDS_MANUAL_REVIEW` and the stage fails closed. `UNKNOWN` is
never promoted to admissible; no self-derived label is treated as
independent.
