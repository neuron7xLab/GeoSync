# Entropy Attestation Method

## Principle
IEV allows hypothesis complexity to expand, then applies deterministic falsification compression.
Output is not intuition but residue after systematic attempted self-destruction.

## Operational phases
1. Entropy expansion: hypothesis, extrapolation, candidate claims.
2. Conflict surfacing: null models, falsifiers, witness signal, purpose drift, external drift.
3. Deterministic compression: contract checks + STN hyperdirect inhibition.
4. Outcome:
   - `KILLED` (negative evidence artifact), or
   - `EVIDENCE` (bounded verified artifact).

## Non-goal
Not proving universal truth; only mechanically rejecting falsity under declared contract.
