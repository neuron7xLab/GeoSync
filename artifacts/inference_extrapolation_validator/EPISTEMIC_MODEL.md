# Epistemic Model

## Fundamental unit of intelligence in this subsystem
**Closed validation cycle**, not an isolated neuron/thought/model/action.

Cycle:
1. intention (select objective)
2. inference (derive within context)
3. extrapolation (project beyond context)
4. falsification (mechanically attempt to break)
5. integration (retain only survived bounded artifact)

## Anti-lying policy
The subsystem never treats extrapolation as truth.
It only emits bounded contractual evidence (`VERIFIED_EVIDENCE`) after required checks pass.
Any violation fails closed.
